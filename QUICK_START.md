## Troubleshooting

**Q: Application won't start**  
A: Make sure Python 3.7+ is installed and requirements are installed

**Q: Styles look broken**  
A: Clear your browser cache (Ctrl+Shift+Delete) and refresh

**Q: PDF won't download**  
A: Check if JavaScript is enabled and try a different browser

**Q: Form data disappeared**  
A: Check browser's incognito/private mode - local storage might be disabled
